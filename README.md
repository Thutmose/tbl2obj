# tcn2obj
Converter from Techne .tcn (xml) models to Wavefront .obj
There is no GUI or CUI, it just converts all .tcn files in working directory.

Required libraries:
 - Apache Commons IO 2.4
 - LWJGJ util 2.9.0
 - Google GSON
 - Google guava
